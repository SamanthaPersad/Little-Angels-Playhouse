# Little-Angels-Playhouse
## Project 4

#### Little Angels Playhouse is a daycare center /after school care program. We have three different classes for kids of varying ages. There is a class for infants, Toddlers and after school kids. Parents can easily navigate through the website and find a particular class for their child depending on their age. They can add their child to a class. They can update the childs information as well as remove their child from a class.

## User Story
#### as a user I want to be able to sign up or log in to my account
#### as a user I want to see all the classes
#### as a user I want to be able to sort a class by the age group
#### as a user I want to be able to search for a class for an infant, toddler, kid
#### as a user I want the number of kids in a class
#### as a user I want to add a child to a class
#### as a user I want to update the number my childs information
#### as a user I want to delete my child from a class
